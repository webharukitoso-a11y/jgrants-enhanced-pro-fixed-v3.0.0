Jグランツから補助金情報を自動取得し、Gemini AIで高品質SEO記事を生成する完全自動化システム

== Description ==

Jグランツ自動投稿システム Enhanced Proは、政府の補助金情報ポータル「Jグランツ」から補助金情報を自動取得し、Google Gemini AIを使用して高品質なSEO記事を自動生成するWordPressプラグインです。

= 主な機能 =

* Jグランツ API から補助金情報の自動取得
* Google Gemini AI による高品質記事の自動生成
* SEOに最適化されたコンテンツの自動作成
* カスタム投稿タイプ「補助金情報」の自動管理
* バッチ処理による効率的な自動化

= システム要件 =

* WordPress 6.0以上
* PHP 7.4以上
* Google Gemini API キー

== Installation ==

1. プラグインファイルをアップロードし、有効化してください
2. 管理画面の「Jグランツ Pro」→「設定」からGoogle Gemini APIキーを設定
3. 自動投稿を有効にして運用開始

== Frequently Asked Questions ==

= このプラグインは無料ですか？ =

はい、プラグイン自体は無料です。ただし、Google Gemini APIの利用料金が別途発生します。

= どのくらいの頻度で記事が生成されますか？ =

デフォルトでは5分間隔で2件ずつ、1日最大100件の処理を行います。
